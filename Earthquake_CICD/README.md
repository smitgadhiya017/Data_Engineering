# Earthquake CI/CD Pipeline

## Project Overview
This project demonstrates a CI/CD pipeline using GitHub Actions, AWS Lambda, and DynamoDB.

## Workflow
1. Fetch data from the USGS Earthquake API
2. AWS Lambda processes the data
3. DynamoDB stores the earthquake records
4. GitHub Actions automatically deploys code to Lambda

## Tech Stack
- Python
- GitHub Actions
- AWS Lambda
- Amazon DynamoDB
- USGS Earthquake API
